# TechTrends Analytics - Bibliometric Study Dashboard

A comprehensive bibliometric analysis dashboard for tracking technology trends in **Blockchain**, **Tokenization**, and **Smart Contracts** from 2020 to 2025. This project collects data from the OpenAlex API and visualizes it through an interactive web-based dashboard.

![Dashboard Preview](https://via.placeholder.com/800x400?text=TechTrends+Analytics+Dashboard)

## üöÄ Features

*   **Automated Data Collection**: Python script (`collect_data.py`) fetches publication counts and top articles from OpenAlex.
*   **Interactive Dashboard**: Modern, dark-mode web interface built with HTML, CSS, and JavaScript.
*   **Key Metrics**: Track total publication volume and growth trends over the last 5 years.
*   **Visual Analytics**:
    *   **Trend Lines**: Visualize the rise or fall of interest over time.
    *   **Volume Comparison**: Compare the total research output across different topics.
*   **Qualitative Insights**:
    *   Explore top 5 most cited articles per year.
    *   **Smart Filtering**: Filter by Topic, Year, and Search by text.
    *   **Keyword Analysis**: Dynamic tag cloud showing the most frequent concepts.

## üìä Usage

### 1. Collect Data
The project comes with a pre-populated `data.json`. To fetch fresh data:

```bash
python collect_data.py
```
*This script will query the OpenAlex API and update `data.json` with the latest publication stats and top articles.*

### 2. View Dashboard
Simply open the `index.html` file in your web browser:

*   **Double-click** `index.html` in your file explorer.
*   OR use a local server (recommended for best performance):
    ```bash
    # Using Python 3
    python -m http.server
    # Then open http://localhost:8000 in your browser
    ```

## üìÇ Project Structure

*   `collect_data.py`: Python script for fetching data from OpenAlex.
